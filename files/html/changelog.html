Changelog

Cryptalis v1.0.0.0 | Released on: 18.07.2025 | The first release

  • A Password Generator, and a Calculator got programmed to the Software.
  • You need to log in, to use the Software.
  • You can also see the Credits of the Software after you logged in.
  • NOTE: On every release of this Software if you type an answer with an spacebar including, and then clicking enter will cause the Software to crash.
  • This bug cannot be fixed, because the Software is made in Windows-Batch (.bat), and was later converted to exe. Windows-Batch dosen't support spacebars as answers.

Cryptalis v1.0.0.1 | Released on: 06.08.2025 | The Console Update

  • A Console got added to the Software.
  • All added console Commands:
    • help = Shows a help page with all avaible commands
    • cls = Clears the Console
    • clear = Clears the Console
    • exit = Exits the Console
    • q = Exits the Console
    • ver = Shows Software version
    • version = Shows Software version
  • The Console can show a help page, clear all its Text, Exit and show the Software version.

Cryptalis v1.0.0.2 | Released on: 30.08.2025 | The Advanced Password Generator Update

  • In the "Advanced Password Generator Update" of Cryptalis the Passwords in the Password Generator got changed from max. 3 Passwords to max. 16 Passwords.

Cryptalis v1.0.0.3 | Released on: 14.09.2025 | The Gaming Launcher Update (part 1)

  • A game called "Quiz Game" original "QuizGame" got added in the Cryptalis v1.0.0.3 Update!
  • These questions are in the new Game:
    • Question 1: Who owns the company "Microsoft"?
    • Question 2: What is 1 + 1?
    • Question 3: What does "perro" mean in spanish?
    • Question 4: Can pigs fly?
    • Question 5: Are cats green?
  • Have fun playing it!

Cryptalis v1.0.0.4 | Released on: 05.10.2025 | The Gaming Launcher Update (part 2)

  • A game called "Guessing game" got added in the Cryptalis v1.0.0.4 Update!
  • Also called as "The Gaming Launcher Update (part 2)".
  • In this game, you have to guess the right number between 1 and 1000000.
  • If the number is too low, then it says "Higher!", if the number is too high, then it says "Lower!".
  • After you win, the game says "Correct!" and underneath if asks you if you want to play it again!
  • And that's it for this Cryptalis Update!
